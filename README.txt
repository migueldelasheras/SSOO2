El proyecto está dividido en un total de 6 ficheros .c, uno por cada tipo de proceso que se puede 
generar durante la ejecución del programa.
Para poder ejecutar el programa, primero deberemos compilar y crear los archivos y directorios 
correspondientes. Para esto, utilizaremos la herramienta make y se deberá ejecutar el comando 
"make all".
Una vez hemos generado todos los archivos necesarios, ejecutaremos una prueba del programa 
ejecutando el comando "make test".
Para realizar una limpieza de todos los archivos generados(salvo el directorio de seguridad "copia"),
deberemos ejecutar el comando "make clean".
